# Changelog

All notable changes to this project will be documented in this file.

## v0.1.0 - 2026-04-06

- introduced the initial public scaffold for a file-based multi-agent control plane
- added worker registry generation from `worker-profile.json`
- added portable `_agent_profile` snapshot generation
- added example local workers, project workers, and subordinate workers
- added tests, GitHub templates, and initial release documentation
